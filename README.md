# Poker-Card-Classification

โปรเจกต์นี้เป็นการจำแนกไพ่โป๊กเกอร์ (Poker Card Classification) โดยการใช้โมเดล Deep Learning (เช่น YOLO) เพื่อจำแนกประเภทของไพ่ตามหน้าไพ่ (Rank) และดอกไพ่ (Suit) ภาพรวมของโปรเจกต์ประกอบไปด้วยไฟล์และโฟลเดอร์ต่างๆ ที่ใช้เตรียมข้อมูลและฝึกสอนโมเดลดังนี้

## 📄 โครงสร้างไฟล์และสคริปต์ (Files)

- **`convert.py`**
  ใช้สำหรับอ่านไฟล์วิดีโอ (จากโฟลเดอร์ `Raw_Video_data`) เพื่อทำการสกัดและแยกภาพเฟรมจากวิดีโอออกมาเป็นรูปภาพ และจัดหมวดหมู่แยกตามหน้าไพ่และดอกไพ่โดยอัตโนมัติ (เช่น นำไปสร้างไว้ใน `pokerdataset/`)

- **`reshape.py`**
  ใช้สำหรับทำ Data Augmentation (การเพิ่มภาพข้อมูล) โดยการอ่านภาพที่แยกไว้ แล้วนำมาแปลงด้วยวิธีต่างๆ เช่น Resize, ขยับภาพ (Translate), สุ่มหมุน (Rotate), และเติม Noise เช่น Salt and Pepper เพื่อให้โมเดลมีความทนทานและแม่นยำขึ้นแล้วบันทึกใน `augmented_poker_data/`

- **`yolo.ipynb`**
  ไฟล์ Jupyter Notebook ที่เป็นส่วนหลักในการรันโค้ดฝึกสอน (Training), การตรวจสอบความถูกต้อง (Validation), และการประเมินผลโจทย์จำแนกภาพด้วยโมเดลสถาปัตยกรรมตระกูล YOLO (You Only Look Once)
